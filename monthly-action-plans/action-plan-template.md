# Monthly Local SEO Action Plan (Template)

Copy this template each month (e.g. `2025-02-action-plan.md`) and fill it in. Keeping a steady monthly rhythm is what compounds local SEO results over time.

## Month: ____

## 1. Google Business Profile

- [ ] Publish 1-2 Google posts (offer, tip, or update).
- [ ] Add new photos.
- [ ] Respond to all new reviews.
- [ ] Answer any new Q&A.
- [ ] Review Insights (calls, directions, searches) and note trends.

## 2. Reviews

- [ ] Ask happy patients for Google reviews this month.
- [ ] Update review counts in review-platforms/review-tracker.md.
- [ ] Sync the review count used in schema (schema-markup repo).

## 3. Citations & NAP

- [ ] Submit / claim any pending directories (see citations/).
- [ ] Spot-check 3-5 existing listings for NAP accuracy.

## 4. Listings

- [ ] Confirm Apple Maps and Bing Places are accurate.
- [ ] Update hours for any upcoming holidays everywhere.

## 5. Rankings & competitors

- [ ] Record current ranks in local-keywords/local-keyword-tracking.md.
- [ ] Note any competitor changes (competitor-notes/).

## 6. Content

- [ ] Coordinate with the blog-content repo on any new local post.
