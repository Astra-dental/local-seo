## Process

1. Search each directory for an existing listing first (avoid duplicates).
2. Claim the existing listing if found; otherwise create one with canonical NAP.
3. Add website, hours, categories, and a description.
4. Record the live URL and the date verified in this tracker.
5. Re-audit quarterly for NAP drift.

## Notes

- Account creation and claiming should be done by the business owner. Do not create accounts on the client’s behalf.
- Remove or merge any duplicate listings found.
- Keep this tracker as the single record of where the business is listed.
