# NAP Consistency Reference

NAP (Name, Address, Phone) must be IDENTICAL across every online listing. Inconsistent NAP confuses search engines and weakens local rankings. Use the canonical values below everywhere.

## Canonical NAP (source of truth)

| Field | Value |
| --- | --- |
| Business Name | Astra Dental Centre |
| Address line | Unit 120, 20061 Fraser Hwy |
| City / Province | Langley, BC |
| Postal code | V3A 0R4 |
| Phone | 604-533-8806 |
| Website | https://astradentalcentre.com/ |
| Email | reception@astradentalcentre.com |

## Formatting rules

- Always write the name exactly as "Astra Dental Centre" (no abbreviations, no "Inc.").
- Use the same unit/suite format ("Unit 120") on every listing.
- Use the local phone number 604-533-8806 consistently (not a tracking number).
- Use the https:// version of the website with a trailing slash.
