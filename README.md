# Astra Dental Local SEO

Workspace for Astra Dental Centre local SEO: Google Business Profile tasks, citation tracking, Apple Maps and Bing Places notes, NAP consistency, reviews, local keywords, competitor notes, and monthly action plans.

Website: https://astradentalcentre.com/

## Purpose

This repository keeps all local-SEO work organized in one place so listings stay accurate and consistent and local map visibility improves over time.

## Repository structure

| Folder | What it holds |
| --- | --- |
| `google-business-profile/` | GBP setup and ongoing task tracker |
| `citations/` | Directory/citation tracker with status per listing |
| `apple-maps/` | Apple Business Connect setup notes |
| `bing-places/` | Bing Places setup notes |
| `nap-consistency/` | Canonical NAP reference (source of truth) |
| `review-platforms/` | Review tracker and response guidelines |
| `local-keywords/` | Local keyword rank tracking |
| `competitor-notes/` | Local competitor observations |
| `monthly-action-plans/` | Monthly action-plan template and logs |
| `published-backups/` | Snapshots of live listings/assets |
